# IT1214-Student-Grade-Systems
🎯 Project Purpose
This project is a simple Java Console-Based Student Grade Management System developed using OOP concepts.
It helps teachers manage student records digitally instead of using paper or spreadsheets.

✨ Features
➕ Add Student
📋 Display All Students
🔍 Search Student by ID
📊 Calculate Average Marks
🚪 Exit System
